# 🏗️ MedChron: System Architecture

**MedChron** is designed as a "Zero-Knowledge" passive sensing platform. It ensures that sensitive behavioral data is processed locally (Edge AI) using **MedGemma 1.5**, with the user's Google Account serving as the secure, centralized synchronization hub.

---

## 🛰️ 1. Data Ingestion Layer
MedChron aggregates "digital exhaust" from three primary sources:

* **Mobile (Android/iOS):** Uses **Google Health Connect** and **Apple HealthKit** to pull device inactivity (sleep proxy), step counts, and screen-time categories.
* **Laptop (Windows/macOS/Linux):** A lightweight background service (Python/Rust) monitors CPU load spikes and "App-Switching Density" (Window focus changes).
* **Central Repository:** All raw time-series data is synced to a private, encrypted folder in the user's **Google Drive/Cloud**, accessible only by the MedChron local instance. 

---

## 🧠 2. The Reasoning Engine (Local Inference)
The core of MedChron is the **MedGemma 1.5 4B Inference Engine** running on the user's hardware (or a dedicated Kaggle/Edge environment).

1.  **Preprocessing:** Raw usage logs are converted into **FHIR-lite** behavioral observations.
2.  **Quantization:** The model uses **4-bit NormalFloat (NF4)** to fit within 4GB - 6GB of VRAM, making it compatible with modern laptops and mobile GPUs.
3.  **Longitudinal Analysis:** Unlike standard LLMs, MedGemma analyzes the *trend*—comparing today's "App-Switching" frequency against the user's 7-day baseline to detect acute stress.

---

## 🔒 3. Privacy & Security Model
* **Edge Processing:** No raw behavioral data is ever sent to a central MedChron server. The "Live Report" is generated locally.
* **Encryption:** Data at rest is secured via **AES-256**.
* **User Authority:** The user can "Purge" logs or "Pause" sensing at any time via the centralized dashboard.

---

## 📊 4. Output: The Live Mental Health Report
The final output is an empathetic, peer-like narrative generated by **Gemini 1.5 Pro**, grounded in the clinical facts provided by **MedGemma**.

* **Circadian Rhythm Score:** Based on device inactivity bins.
* **Cognitive Load Rating:** Based on multitasking density.
* **Burnout Alert:** A predictive marker based on sustained high-intensity device usage.
